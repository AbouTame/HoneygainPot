# Cách để có mã lỗi/nhật ký báo lỗi?
1. Vào tab Actions trên repo đã fork của bạn
2. Chọn tiến trình chạy bị lỗi
3. Chọn tab `Manual` hoặc `Daily` rồi chọn tab `Run HoneygainPot` bạn sẽ thấy mã lỗi/nhật ký báo lỗi
